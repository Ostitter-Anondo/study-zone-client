<h2>🧐 Features</h2>

Here're some of the project's best features:

*   Dynamic user handling
*   Beautified by Tailwind CSS and daisyUI
*   Write Submit Read Edit and Delete videogame reviews
*   Maintain a watchlist for games to look out for
